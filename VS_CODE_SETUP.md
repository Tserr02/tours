# Запуск проекта в Visual Studio Code

1. Откройте папку `C:\Users\zerr_\Desktop\Tours` в Visual Studio Code.
2. Установите рекомендованные расширения, если VS Code предложит это сделать.
3. Откройте меню `Terminal` -> `Run Task...`.
4. Выполните задачи по порядку:
   - `1. Создать виртуальное окружение`
   - `2. Установить зависимости`
   - `3. Запустить PostgreSQL`
   - `4. Применить миграции`
   - `5. Заполнить демо-данные`
   - `6. Создать администратора`
   - `7. Запустить сайт`

После запуска сайт будет доступен по адресу:

```text
http://127.0.0.1:8000/
```

Административная панель:

```text
http://127.0.0.1:8000/admin/
```

Проект использует отдельное окружение `.venv-vscode`, чтобы не конфликтовать со старой папкой `.venv`, если она уже занята VS Code.

Если Docker не установлен, PostgreSQL нужно установить отдельно и создать базу с параметрами из файла `.env`.
